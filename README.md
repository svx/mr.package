# mr.package
dirty wrapper around fpm for package building

## Features

- easy package building
- easy to change default settings

## Dependencies
[fpm](https://github.com/jordansissel/fpm)

## Examples

	./build_deb.bash -m sven@so36.net -n $MYPACKAGE -p /srv/zope

For help:

	./build_deb -h
